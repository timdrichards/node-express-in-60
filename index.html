Welcome Node & Express in 60 Minutes

In this workshop we will explore the basics of web development with Node & Express in 60 minutes. We will introduce the fundamentals of using Node and the infamous Express web application framework through a series of examples. This is a hands-on workshop so boot up your system and get ready!

The slides for this workshop can be found here.

There are three parts to this workshop. Each part is maintained as a separate branch on github.
